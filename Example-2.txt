2. Объясните почему блок "Work 2" не выполняется? let serialQueue = DispatchQueue(label: "com.foo.bar")
serialQueue.async { print("Work 1")
 serialQueue.sync { print("Work 2")
} }
 
Ответ: Происходит так называемый deadlock.То есть идет выполнение последователньой очереди и в асинхронном методе он пытается выполнить синхронный,
это и привело к блокировке, заблокировав ресурсы для "Work 1", пытается выполнить и заблокировать ресурсы для "Work 2" - это вызывает блокировку.
Один из вариантов - использовать DispatchQueue.global() ("параллельную" очередь), либо можно попробовать использование приоритезации.
